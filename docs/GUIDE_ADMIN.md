# Guide Utilisateur Administrateur - TruckFlow

## Connexion
- **URL** : Accédez à l'application TruckFlow
- **Identifiants** : Email + mot de passe administrateur
- **Première connexion** : Changez votre mot de passe

## Tableau de bord
### Vue d'ensemble
- **Statistiques** : Camions totaux, trajets actifs, maintenance en attente
- **Actions rapides** : Créer trajet, ajouter camion, programmer maintenance
- **Activité récente** : Dernières actions système

## Gestion de la flotte

### Camions
1. **Ajouter** : Menu Camions > Ajouter camion
   - Plaque d'immatriculation (obligatoire)
   - Marque, modèle, année
   - Capacité, kilométrage, statut
2. **Modifier** : Clic sur icône ✏️
3. **Filtrer** : Par statut, recherche par plaque/marque

### Remorques
- **Types** : Plateau, Frigorifique, Citerne, Conteneur
- **Informations** : Plaque, type, capacité, kilométrage

### Pneus
- **Suivi** : Numéro série, position, état d'usure
- **Gestion** : Rotation préventive, remplacement

## Gestion des conducteurs
1. **Ajouter** : Informations personnelles + permis
2. **Permissions** : Conducteur/Superviseur/Administrateur
3. **Suivi** : Trajets, performances, formations

## Gestion des trajets
### Création
1. Menu Trajets > Ajouter trajet
2. Sélectionner : Camion, conducteur, itinéraire, date
3. **États** : En attente → En cours → Terminé/Annulé

### Suivi
- Position GPS, kilométrage, consommation
- Commentaires conducteur, incidents

## Maintenance
### Types
- **Préventive** : Vidange (15k km), pneus (10k km), freins (20k km)
- **Corrective** : Réparations, remplacements

### Programmation
1. Menu Maintenance > Ajouter maintenance
2. Type, véhicule, date, description, coût
3. **Statuts** : Programmée → En cours → Terminée

## Rapports et analyses
### Tableaux de bord
- **Carburant** : Consommation mensuelle, coûts
- **Trajets** : Répartition par statut, distances
- **Maintenance** : Coûts par type, fréquence
- **Flotte** : Disponibilité, utilisation

### Export
- PDF (rapports formatés)
- Excel (données brutes)
- Filtrage par période
